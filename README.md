# Todo App

React + TypeScript + Tailwind CSS で作ったシンプルな TODO アプリです。

## 機能

- **タスク管理** — 追加・編集・削除・完了チェック
- **優先度** — 高 / 中 / 低 をカラーバッジで表示
- **期日** — 設定可能、期限切れは赤でハイライト
- **フィルター** — すべて / 未完了 / 完了
- **ソート** — 作成日 / 優先度 / 期日 / タイトル
- **検索** — タイトル・説明文をリアルタイム検索
- **ダークモード** — OS 設定に連動、手動切替も可能
- **永続化** — LocalStorage にデータを自動保存

## 使い方

```bash
npm install
npm run dev
```

`http://localhost:5173` をブラウザで開く。

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Ctrl + Enter` | タスクを保存（編集モード） |
| `Esc` | 編集をキャンセル |
